Sync to Google Sheets

Push your data to Google Sheets directly using the Sheets API. Fast, reliable — handles 3 lakh+ rows in minutes.

Two things needed (one-time setup):
1. Sheet ID — from your Google Sheet URL
2. Apps Script URL — for state team search (same as before)

Sync uses Google Sheets API directly — no row limits, no timeouts, much faster.

Settings

Step 1 — Connect Google Account (for fast sync via Sheets API)

Sign in with the Google account that owns the Sheet. Enables direct write — handles 3 lakh rows in ~3 minutes. Skip this if using Apps Script fallback.

Not connected
No Client ID yet? Go to console.cloud.google.com → APIs & Services → Credentials → Create OAuth 2.0 Client ID → Web Application → add https://cscspm.github.io as Authorized JS origin → copy Client ID

Step 2 — Sync Data to Google Sheets

Writes all data directly. With Google Account connected: 3 lakh rows in ~3 minutes. Without: uses Apps Script fallback (slower but works from local file too).

Backup & Restore

Export all your stored data as a single backup file. Import it on any device or browser to restore everything instantly.

Step 1 — Export

Download Backup Files

Exports your data as multiple small .json files (one per dataset). This prevents browser crashes with large data.

You will get 4–8 files — save all of them in one folder on Google Drive.

📁 Files downloaded: _1_meta_history.json · _2_master.json (may split into parts) · _appstage.json · _ekyc.json

Import Backup Files

Import each backup file one at a time. Data is merged — import all files to fully restore.

Start with _1_meta_history.json, then master parts, then appstage, then ekyc.

Click or drag · one .json file at a time
How to Use on Mobile / Another Device
①
Export backup on this device
Click "Export All Data" above. A .json file downloads to your laptop.
②
Send to yourself
Upload the .json to Google Drive, WhatsApp to yourself, or email it to your phone.
③
Open the HTML tool on mobile
Open the csc_processor_v7.html file from Google Drive in Chrome on your phone.
④
Import the backup
Go to Backup & Restore → Import the .json file. All data restored in seconds.
What is included in the backup:
✓ Master Blocked/Unblocked IDs (all entries)
✓ App Stage data (all CSC IDs + stage numbers)
✓ EKYC data (all unique IDs + dates)
✓ Fraud IDs (all flagged entries with details)
✓ Unblock History (all dates you processed IDs)
✓ Permanently Closed history (all dates)

What is NOT included: Today's daily verified/non-verified uploads (those are session-only, upload fresh each day).
